# 🎓 Edulance – AI Powered Student Freelancing Platform

> A modern marketplace where students can post academic tasks, find skilled peers, collaborate securely, and receive intelligent task assistance powered by AI.

---

# 🚀 Problem Statement

Students frequently struggle to find trustworthy peers for assignments, presentations, coding projects, reports, notes, and academic assistance.

Existing freelancing platforms are:
- Too expensive
- Not student-focused
- Difficult for beginners
- Lack academic-specific features

Edulance solves this by creating a dedicated marketplace built specifically for students.

---

# 💡 Solution

Edulance connects students who need academic help with students willing to complete tasks.

The platform provides:

- Secure Authentication
- Real-time Chat
- Task Marketplace
- Student Profiles
- Applications System
- Notifications
- File Sharing
- Modern Mobile UI
- AI-assisted Task Analysis (Gemma integration in progress)

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/vinit130/Edulance.git
```

Install Dependencies

```bash
npm install
```

Run Metro

```bash
npx expo start
```

Run Android

```bash
npx expo run:android
```
